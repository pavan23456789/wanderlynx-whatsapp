# **App Name**: Travonex WhatsApp Ops Dashboard

## Core Features:

- Authentication: Secure dashboard with email/password login and role-based access control (Super Admin, Internal Staff).
- WhatsApp Webhook Handler: Handles WhatsApp webhook events (message received, status updates) and ensures webhook verification.
- Contact Management: Create, view, and manage contacts with details like phone, name, email, trip details. Auto-create contacts on incoming messages and skip duplicates.
- Excel/CSV Contact Upload: Super Admins can upload contacts in bulk via Excel/CSV files, with validation and duplicate skipping.
- Contact Export: Super Admins can download contacts as CSV files.
- Inbox and Manual Reply: View conversations, read messages, and manually reply to users within the 24-hour WhatsApp window, with template-based replies when the window is closed.
- Message Template Management: Create, manage, and use message templates for campaigns and replies.
- Campaign Messaging: Send broadcast messages/campaigns to selected contacts using templates, with basic status tracking (sent, delivered, failed).
- Basic Automation: Automatic reply on the first incoming message.

## Style Guidelines:

- Primary color: Soft blue (#90CAF9) to convey trust and reliability, aligning with professional communication.
- Background color: Light gray (#F5F5F5) to ensure a clean and uncluttered workspace, preventing eye strain during long hours of use.
- Accent color: Subtle teal (#4DB6AC) to draw attention to important CTAs without overwhelming the UI.
- Body and headline font: 'PT Sans', a humanist sans-serif for readability and a modern, warm feel.
- Use a consistent set of minimalistic icons from a library like Material Icons to maintain a clean and intuitive interface.
- Prioritize a clear, tabular layout for data display, focusing on information hierarchy to improve usability.
- Employ subtle animations (e.g., loading spinners, button hover effects) to provide feedback and enhance the user experience without being distracting.
